Feeder
==========
Feeds curated by your friends.

Initial dev version in progress.

Getting Started
---------------
Requires a running instance of MongoDB.

Clone and setup:

    $ git clone https://github.com/retric/feeder.git myproject

    $ cd myproject

    # Install dependencies
    $ npm install

    $ node app.js
